The Logic

Not null. Not undecided.
Deliberating.

Binary forces every calculation into YES or NO. Ternary gives agents the essential third option allowing them to state insufficient signal and trigger deliberate evidence gathering.

-1

Reject

Signal is negative and resolvable. Evidence definitively points away from action. Do not proceed. This is a computed decision, not a system error.

conflict · signal negative
0

Tend

The deliberation zone. Neither null nor indecision. An active instruction to the VM: gather more contextual evidence before committing weights.

hold · active computation
+1

Affirm

Signal is affirmative. Evidence supports action. Proceed with execution, generating a confidence score that quantifies structural alignment.

truth · signal affirmative
